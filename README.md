# Sales Performance Analysis (MySQL)

## Project Overview
This project analyzes retail sales data using MySQL to understand revenue trends,
product performance, and customer purchasing behavior.

## Business Problem
The business wants to identify:
- Top revenue-generating products
- High-value customers
- Monthly sales trends to support better decision-making

## Tools & SQL Features Used
- MySQL
- Aggregations (SUM, COUNT)
- Date functions
- Subqueries
- Window functions (RANK)

## Key Analysis Performed
- Total and monthly revenue analysis
- Top products and customers by revenue
- Repeat vs one-time customer analysis
- Revenue ranking using window functions

## Key Insights
- A small set of products contributes most of the revenue
- Repeat customers generate higher total revenue
- Sales show clear monthly trends indicating seasonality

## Business Recommendations
- Focus marketing on high-performing products
- Introduce loyalty programs for repeat customers
- Plan inventory based on monthly sales trends
